## Запуск

Кожен ноутбук самодостатній. Відкривайте в JupyterLab або VS Code:

```bash
pip install jupyterlab pandas numpy scikit-learn matplotlib seaborn torch tensorflow
jupyter lab
```

Деякі папки містять власний `requirements.txt` для специфічного датасета / фреймворку, потрібного на цьому тижні.

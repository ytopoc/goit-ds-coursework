# GoIT Data Science Coursework

Thirteen weekly homework assignments from a Data Science course taken May–August 2024. Each folder is one week's lesson, named `Hw{N} {DD.MM.YYYY}`.

## Topics covered

| Folder | Topic |
|--------|-------|
| `Hw1.ipynb` | Python + NumPy fundamentals |
| `Hw2 19.05.2024/` | Pandas — data exploration on bestsellers and survey data |
| `Hw3 27.05.2024/` | Linear regression on housing prices |
| `Hw4 05.06.2024/` | Overfitting & regularization (bike-rental dataset) |
| `Hw5 11.06.2024/` | Classification — logistic regression, kNN |
| `Hw6 21.06.2024/` | Clustering — k-means on 2D and MNIST |
| `Hw7 30.06.2024/` | Decision trees & random forests |
| `Hw8 06.07.2024/` | Recommender systems |
| `Hw9 09.07.2024/` | NLP basics — tokenization, TF-IDF |
| `Hw10 19.07.2024/` | Convolutional networks + transfer learning (VGG) |
| `Hw11 26.07.2024/` | Time series / sequence models |
| `Hw12 01.08.2024/` | Generative models |
| `Hw13 03.08.2024/` | Final project — CNN model + a small Streamlit app (`my_app.py`) |

## Running

Each notebook is self-contained. Open in JupyterLab or VS Code:

```bash
pip install jupyterlab pandas numpy scikit-learn matplotlib seaborn torch tensorflow
jupyter lab
```

Some folders contain their own `requirements.txt` for the specific dataset / framework needed that week.
